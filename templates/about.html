NotesTrack

NotesManagement is a simple and secure web application that allows users to create, store, update, and manage notes efficiently.

This application is built using Flask, Python, MySQL, and Bootstrap. It focuses on simplicity, security, and ease of use.

Users can register, log in securely, add new notes, edit existing notes, and delete notes whenever required. Each user can access only their own notes, ensuring privacy and data security.
